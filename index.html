HTML & CSS Basics Summary

HTML Summary

HTML(hypertext markup language) is used to define our page content, structureand meaning. you don't use it for styling purposes. use CSS for that instead.

  • HTML uses 'elements' to describe(annotate) content
  • HTML elements typically have an opening tag and then a closing tag
  • you can also have void(empty) elements like images
  • you can also configure elements with attributes
  • there's a long list of available elements but you'll gain experience overtime, no worries.

learn more about HTML elements on the MDN HTML elements reference

CSS Summary

CSS(cascading style sheet) is used for styling your page

  • styles are assigned via property-value pairs
  • you can assign styles via 'style' attribute
  • to avoid code duplication, you typically use global styles
  • alternatively, you can work with external stylesheet files which you link to
  • when working with CSS concepts like 'inheritance', 'specificity' and 'the box model' should be understood

    • learn more about HTML elements on the MDN HTML elements refrence
